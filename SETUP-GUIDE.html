Local AI – Oppsett (Windows, Firmaportal)

Denne siden forklarer hvordan du installerer det som trengs for å kjøre Local AI på en skole-PC. Oppsettet krever ikke admin-rettigheter, og etter installasjon kan AI-en brukes uten internett.

Viktig: Hvis dere bruker Firmaportal for Python, får du ofte ikke valg om PATH. Hvis du får valget “Add Python to PATH”, er det trygt å la det være AV. Programmet fungerer uansett.

Forutsetninger

  • Windows-PC
  • Python installert via Firmaportal
  • Internett kun for installasjon av pakker (én gang)
  • Anbefalt: minst 6–8 GB RAM

Steg 1: Installer Python via Firmaportal

  1. Åpne Firmaportal
  2. Finn Python 3.12 (eller nyere 3.x-versjon)
  3. Klikk Installer
  4. Vent til installasjonen er ferdig
Om “Add Python to PATH”: Hvis du får dette valget, la det gjerne være AV. Det er ikke nødvendig for å bruke Local AI.

Steg 2: Åpne PowerShell

  1. Trykk Windows-tasten
  2. Skriv PowerShell
  3. Trykk Enter

Du får et blått vindu med tekst. Det er normalt.

Steg 3: Installer nødvendige Python-pakker

Kjør kommandoene under én og én i PowerShell.

Installer Torch

pip install --user torch

Installer Transformers

pip install --user transformers
Tips: Hvis PowerShell sier at pip ikke finnes, lukk PowerShell og åpne det igjen. Prøv på nytt.

Steg 4: Start Local AI

  1. Åpne mappen LocalAI
  2. Dobbeltklikk på filen LocalAI_GUI.py

Du trenger ikke PowerShell for å starte programmet etter at pakkene er installert.

Bruk

  1. Velg modell øverst (for eksempel TinyLlama Math)
  2. Klikk Load model
  3. Skriv et spørsmål og trykk Send

Eksempler: “Hva er (-5/4) - (-5/10)?”, “Forklar brøkregning kort”.

Lagring og lasting av chat

  • Save chat lagrer samtalen som en JSON-fil
  • Load chat åpner en tidligere lagret samtale
  • Når du lukker programmet, kan du bli spurt om å lagre hvis chatten ikke er lagret

Feilsøking

  • Kontroller at Python faktisk er installert via Firmaportal
  • Lukk og åpne PowerShell på nytt hvis installasjonen feiler
  • Start PC-en på nytt hvis noe henger
  • Sjekk at modell-mappene ligger i samme mappe som LocalAI_GUI.py
Denne HTML-filen kan åpnes direkte i nettleser. Den gjør ingen endringer på PC-en.
